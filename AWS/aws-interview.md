**Question 1:** A team wants to process images as soon as they are uploaded to Amazon S3. The service should resizes the images automatically. Which of the following AWS service might be most cost-efficient option to build this solution?

1. AWS ECS
2. AWS Lambda
3. AWS EC2
4. AWS EKS

**Answer-** AWS Lambda

**Question 2:** You want to host an API based web services to convert the user provided video file from one format to another. It takes up to 30 mins for single video to be processed. For this, you have exposed your APIs using AWS API gateway. Your developers have built the python based app which does the job and needs to be hosted in the backend. Which of the following AWS service do you think will be most appropriate for this use case?

1. AWS Lambda
2. AWS EC2
3. AWS ECS with Fargate
4. AWS CloudFront

**Answer-** AWS ECS with Fargate

**Question 3:** How does AWS charge for AWS Lambda usage once the free tier has been exceeded?
1. By the amount of memory allocated and the time it takes for the lambda function to run
2. By the number of versions
3. By the number of requests made for a given lambda function
4. By the programming language
5. By the total number of lambda functions in the account

**Answer-** By the amount of memory allocated and the time it takes for the lambda function to run
