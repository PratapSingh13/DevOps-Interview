# About Proxy and Proxy Servers

- A proxy acts as an intermediary between a client (your machine) and a server (the website you're trying to access), it forwards your requests to the server and returns the response to your machine. 

**Proxy Advantages**

- Provides privacy and control by hiding client identities and managing the outbound traffic.
- Security from Malicious Attackers.
- Logging of client activities.
- Performance (By caching)

- A reverse proxy  is an intermediary that sits in front of servers and forwards client requests to the appropriate server.

**Reverse Proxy Advantages**

- Load Balancing
- Security from Malicious Attackers.
- Failover and redundancy
- Caching
- SSL Termination
